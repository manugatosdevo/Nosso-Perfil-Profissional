# atividades-profile

Atividade Prática – Laboratório Web 2 (HTML + CSS)

Tema: Criando uma Página de Perfil Profissional

Objetivo: Aplicar conceitos de HTML e CSS para criar uma página web organizada e visualmente agradável.

Requisitos da Página

Crie uma página contendo:

1. Cabeçalho
Nome do aluno
Título da página: "Meu Perfil Profissional"
2. Sobre Mim
Um parágrafo falando sobre você.
Formação ou curso que está realizando.
3. Habilidades

Crie uma lista com pelo menos 5 habilidades:

Exemplo:
HTML
CSS
Redes de Computadores
Trabalho em Equipe
Informática
4. Contato
E-mail
Telefone
Cidade
5. Rodapé
Frase motivacional ou profissional.
Requisitos de CSS

Aplicar:

✅ Cor de fundo na página

✅ Cor diferente para o cabeçalho

✅ Fonte personalizada (Arial, Verdana ou outra)

✅ Centralizar o título principal

✅ Adicionar bordas em alguma seção

✅ Efeito hover em um botão
